# WTF is My Gear Score?

## Setup

```
bin/setup
```

## Starting the app

First, make sure Postgres and Redis are running.

Then:

```
bin/guard
```

and visit `http://127.0.0.1:3000`

## Running the tests

```
bin/rspec
bin/rake rubocop
```
